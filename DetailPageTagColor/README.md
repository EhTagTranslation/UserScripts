详情页标签颜色
=========

[安装脚本](//sleazyfork.org/scripts/415728);

E站推出的"我的标签"功能非常好用,但是漫画详情页却不像列表一样显示自定义的标签颜色.

**需要先点开我的标签页,功能才会生效**

将标签颜色填充到网页图标中,方便在打开大量页面时候快速查找.
